# Couriway's Minecraft Speedrun Analysis (Power BI)

## 📊 Project Overview

This project analyzes a **1,000 Minecraft speedruns** completed by **Couriway** using **Power BI**.

The dashboard explores different aspects of each run, including progression, performance, deaths, routing decisions, bastion types, and completion statistics. The goal is to demonstrate data visualization, data modeling, and dashboard design using Power BI.

---

## Dataset

The dataset contains **1,000 filtered speedruns** with information such as:

- Date Played
- Seed
- Iron Source
- Enter Type
- Gold Source
- Nether Entry
- Bastion
- Fortress
- Nether Exit
- Stronghold
- End
- In-Game Time (IGT)
- Blaze Rods
- Death Count
- Blaze Kills
- Enderman Kills
- Iron Golem Kills
- Bastion Type
- End Fight Type
- Frame Eyes
- Notes

---

## Dashboard Features

The Power BI dashboard includes visualizations such as:

- 📈 Run performance over time
- ⏱ Average In-Game Time (IGT)
- ☠ Death statistics
- 🏰 Bastion Type distribution
- 🔥 Blaze Rod analysis
- ⚔ End Fight Type analysis
- 📅 Runs by date
- Interactive slicers and filters
- KPI cards summarizing important statistics
